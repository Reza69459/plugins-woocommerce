پرداخت اینترنتی وجه به وسیله درگاه پرداخت واسطه پی پینگ برای ووکامرس فارسی
== Description ==
با استفاده از این افزونه میتوانید دروازه پرداخت بانکی را به فروشگاه اینترنتی خود ایجاد کنید.

= ویژگی ها =
* امکان تعیین واحد پولی به ریال و یا تومان
* سازگاری با نسخه های مختلف وکامرس

== Installation ==

1. پوشه 'payping-woocommerce' را در مسیر '/wp-content/plugins/' آپلود کنید
2. افزونه را از طریق منوی 'افزونه ها' در وردپرس فعال کنید
3. تنظیمات افزونه را می توانید از طریق قسمت تنظیمات ووکامرس / پیکربندی/ درگاه های پرداخت انجام دهید
